# Campus Cliques

A marketing website for Campus Cliques — connecting brands with college ambassadors.

## Files
- `index.html` — main page
- `style.css` — all styles
- `script.js` — form submission logic
- `images/logo-dark.png` — dark background logo (used in navbar)
- `images/logo-white.png` — white/lavender logo (used in hero)

## To run
Open `index.html` in any browser. No build step required.
